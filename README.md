# SamHangSi Generator

SamHangSi generator based on Kor-char-rnn implements by tensorflow

## Example

### Command

```bash
python main.py --word 삼행시
```

- 삼행시

```
삼이야 그리움이 좇아 사랑은늘 도망가
행른 잊어버리고 그대 이 세상
시제 너의 곁을 떠나면 빗물에 꽃씨하나 흘러가듯
```

- 기계

```
기를 바라보네 두 손 잡고 고개 끄덕여 달라 하기에
계 울고 싶어 내 맘을 떠나가던 날
```

- 여름

```
여도 지금하럼 커피는 날개니
름다웠던 그대모습 다시 볼 수 없는것 알아요
```

- 커피

```
커나가 그래 돌아서 눈 감으면 잊을까
피고 내가 가고 싶지 아파 만날 날 기다려왔어
```
